# 表单的 CSS 设置

## textarea

文本框（textarea）用于让用户输入大量文本。它的高度通常是固定的，如果用户输入的文本多于文本框的高度，文本框就会出现滚动条，只有拖动滚动条，才能看见文本的其余部分。

`form-sizing`属性让文本框具有自适应高度，随着用户输入的文本增加，而增加文本框的高度。

```css
textarea {
  form-sizing: normal;
}
```
